Från det att druvan börjar växa tills vinet hälls upp i ditt glas påverkar många faktorer vinets slutliga smak.
Generellt sett tenderar dessa faktorer att grupperas i tre kategorier som var och en motsvarar ett stadium i vinets liv:

<PageBreak />

## Primära aromer

Dessa aromer kommer direkt från druvorten och terroiren (miljön där druvorna odlas).

| **Källa** |                **Aromfamilj**                 | **Exempel**                                          |
| :-------: | :-------------------------------------------: | :--------------------------------------------------- |
|   Druva   |   <Aroma type="citrus">Citrusfrukt</Aroma>    | Citron, Apelsin, Grapefrukt                          |
|   Druva   |  <Aroma type="tree-fruit">Grön frukt</Aroma>  | Äpple, Päron, Krusbär, Vindruva                      |
|   Druva   |   <Aroma type="red-fruit">Röd frukt</Aroma>   | Jordgubbe, Hallon, Röda vinbär                       |
|   Druva   | <Aroma type="black-fruit">Svart frukt</Aroma> | Plommon, Björnbär, Svarta vinbär, Blåbär             |
|   Druva   |  <Aroma type="stone-fruit">Stenfrukt</Aroma>  | Persika, Aprikos                                     |
|   Druva   | <Aroma type="tropical">Tropisk frukt</Aroma>  | Ananas, Mango, Banan                                 |
|   Druva   |     <Aroma type="floral">Blommig</Aroma>      | Blomning, Ros, Viol                                  |
|   Druva   |      <Aroma type="herbal">Örtig</Aroma>       | Mynta, Eukalyptus, Timjan, Rosmarin, Fänkål, Lakrits |
|   Druva   |    <Aroma type="vegetal">Grönsakig</Aroma>    | Paprika, Gräs, Tomatblad, Sparris, Oliv, Rödbeta     |
|   Druva   |      <Aroma type="earthy">Jordig</Aroma>      | Sten, Mineraler                                      |

## Sekundära aromer

Dessa aromer utvecklas under vinframställningsprocessen, främst genom jäsning och på grund av påverkan från ekfat.

|      **Källa**      |            **Aromfamilj**             | **Exempel**                                                         |
| :-----------------: | :-----------------------------------: | :------------------------------------------------------------------ |
| Alkoholisk jäsning  | <Aroma type="microbio">Jästig</Aroma> | Kex, Bröd, Rostat bröd, Deg, Ost, Yoghurt                           |
| Malolaktisk jäsning | <Aroma type="microbio">Krämig</Aroma> | Smör, Ost, Grädde                                                   |
|         Ek          |    <Aroma type="oaky">Ekig</Aroma>    | Vanilj, Kokos, Rök, Kryddnejlika, Ceder, Sågspån, Dill, Blyertsspån |
|     Ekrostning      |  <Aroma type="earthy">Jordig</Aroma>  | Rök                                                                 |

## Tertiära aromer

Dessa aromer utvecklas när vinet mognar på flaska eller i ekfat under en längre tid.

|        **Källa**        |              **Aromfamilj**               | **Exempel**                           |
| :---------------------: | :---------------------------------------: | :------------------------------------ |
|      Flasklagring       |    <Aroma type="earthy">Jordig</Aroma>    | Läder, Jord, Svamp, Skogsmark         |
|      Flasklagring       |   <Aroma type="animal">Djurisk</Aroma>    | Umami, Vilt, Kött, Sojasås, Balsamico |
|      Flasklagring       |      <Aroma type="oaky">Ekig</Aroma>      | Choklad, Kaffe, Tobak                 |
| Flasklagring (vitt vin) |    <Aroma type="earthy">Jordig</Aroma>    | Honung                                |
|    Oxidativ lagring     | <Aroma type="raisin">Torkad frukt</Aroma> | Russin, Fikon, Katrinplommon          |
|    Oxidativ lagring     |    <Aroma type="ageing">Nötig</Aroma>     | Mandel, Hasselnöt, Valnöt             |
|    Oxidativ lagring     |      <Aroma type="oaky">Ekig</Aroma>      | Karamell                              |
